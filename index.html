Calculadora de Média

Calculo de Média Aritimetica Simples.

A calculadora de média é usada para calcular o valor médio de um conjunto de números.

Como calcular o valor médio

Suponha que tenhamos um espaço de amostra a1, a2, ..., an. Em seguida, o valor médio é definido através da seguinte equação:

Média = (a 1 + a 2 + ... + a n ) / n
